# ArcadeDisasters+

ArcadeDisasters+ is a Fabric client-side mod designed for the new Hypixel "Disasters" arcade game. It just provides a few simple QOL features that I wanted while playing.

## Features

- **Quick Join**: Includes a `/disasters` command to instantly join a "Hypixel Disasters" game from anywhere!
- **Disaster Start Notifications**: Display a popup on screen when disasters start, including their descriptions. Keep your eyes on the action :)
- **Disaster End Notifications**: There is almost no indication when a disaster ends, and I would often miss it, so just like the start notifications, a popup appears when disasters end.
- **Settings**: If you don't want some of the features, you can disable/enable them using the **Mod Menu** mod.

## Installation

All you really need is the **Fabric API** and the **Fabric Loader**. If you want settings you'll need **Mod Menu**. The mod is built for Minecraft **1.21.11**.

### Requirements
- **Minecraft**: `1.21.11`
- **Fabric Loader**: `0.18.4` or higher
- **Fabric API**: `0.141.3+1.21.11` or higher
- **Mod Menu** (Optional): `17.0.0-beta.2` or higher (to access the settings screen)
